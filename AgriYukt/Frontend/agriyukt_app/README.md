# Crop Marketplace App

This is a Flutter-based agricultural marketplace application.

Features:
- Buyer and Farmer roles
- Crop ordering system
- Real-time order updates
- Wallet system using Razorpay (UPI, Card)
- Supabase backend

This project is currently under development for learning and hackathon purposes.# agriyukt_app
